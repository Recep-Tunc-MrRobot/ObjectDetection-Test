-----------------------------------------------------------------------------------------------

This project performs real-time object detection using Ultralytics YOLO with a webcam.
To improve performance, the model runs every few frames.

Requirements

Python 3.8+

Webcam

Installation

pip install -r requirements.txt

Usage

python main.py

Note: The best.pt file must be located in the same directory as main.py.
